Site institucional premium da **ANIMA Wellness Fight Club** — academia de musculação, artes marciais e performance em São Paulo.

```

├── index.html          # Página principal
├── assets/

│   ├── anima-premium.css
│   ├── anima.js        # Interações
│   ├── hero-bg.mp4     # Vídeo do hero

```

## Publicar no GitHub

Se ainda não enviou o código:

```bash
cd C:\Users\FIC\Downloads\ANIMAWELLNESS
git remote add origin https://github.com/abarakus11/ANIMAWELLNESS.git
git branch -M main
git push -u origin main
```

## GitHub Pages

1. No repositório, vá em **Settings → Pages**
2. Em **Source**, selecione **Deploy from a branch**
3. Branch: **main** · Pasta: **/ (root)**
4. O site ficará em: `https://abarakus11.github.io/ANIMAWELLNESS/`

## Desenvolvimento local

Abra `index.html` no navegador ou use um servidor local:

```bash
npx serve .
```
